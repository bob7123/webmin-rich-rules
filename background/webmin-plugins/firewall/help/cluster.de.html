Cluster-Firewall-Server
Diese Seite kann verwendet werden, um Ihre Firewall-Konfiguration gleichzeitig auf mehreren Systemen anzuwenden, sodass Sie ein Cluster von Firewall-Systemen über eine einzige Schnittstelle verwalten können. Bevor ein Server zu dieser Seite hinzugefügt werden kann, muss Webmin 1.170 oder höher installiert und im Webmin Servers Index-Modul hinzugefügt sein. Es müssen auch die notwendigen ipi(6)tables Programme installiert und richtig konfiguriert sein.
