Match Element
The primary match element for this rule. Elements are mutually exclusive — you can only select one. Options include:
Service — Match a named service (e.g. http, ssh).
Protocol — Match a specific protocol by name or number.
ICMP Block — Block a specific ICMP message type.
ICMP Type — Match a specific ICMP message type.
Masquerade — Enable source NAT masquerading.
Forward Port — Forward incoming traffic to another port or address.
(none) — No element; rule matches by source, destination, or port only.
