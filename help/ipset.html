IP Set
A firewalld IP set is a named collection of IP addresses, networks, or MAC addresses managed by the ipset tool or firewalld. Instead of creating hundreds of individual rules for each IP, you define a set (e.g. blocklist) containing all the IPs, then reference it by name in a single rule.

To see defined IP sets, run: firewall-cmd --get-ipsets
To create one: firewall-cmd --new-ipset=mylist --type=hash:ip --permanent
To add IPs: firewall-cmd --ipset=mylist --add-entry=192.168.1.10 --permanent
